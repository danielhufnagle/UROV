# UROV Spring 2026

This is our current list of goals for this quarter. These don't need to be followed in exact order, but we should aim for all of these to be done as fast as possible.

1. Get 3 dimensional twist generation from joystick input working
2. Get ROS2 running on the raspberry pi (flash Ubuntu 24.04 onto it)
3. Drive motors with ROS (be prepared to drive ESCs directly off of the Pi, as the flight controller might not get here in time)
4. Drive motors according to the joystick input
5. Be able to take a video with an onboard camera
6. Be able to measure the coral garden somehow (doesn't necessarily have to be with photogrammetry, but should figure it out somehow)
