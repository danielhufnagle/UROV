# NURC UROV

### Requirements:
- VSCode
- Docker

### Setup:
1. In VSCode, install the "Remode Development" extension. This will allow you to run the development environment
2. Click on the little arrows symbol in the bottom left corner of VSCode.
3. Click "Reopen in Container"
You should now be in the docker container with ROS 2 installed.
